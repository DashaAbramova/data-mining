## 👨‍🏭 Советчик сталевара

**Цель проекта:**  
Разработать цифрового советчика, который рекомендует сталевару точное количество и наименование ферросплавов для достижения целевого химического состава стали.

**Бизнес-контекст:**  
На предприятии сталь проходит путь:  
`Руда → Доменная печь (чугун) → Дуговая печь (полупродукт) → Печь-ковш (легирование)`  

На участке «печь-ковш» в полупродукт добавляют ферросплавы. Нужно, чтобы итоговое содержание элемента (`Элемент_Final`) попало в интервал `(Элемент_LowerLimit, Элемент_Target)`.

### 📊 Данные (Excel)

Проект включает обработку 10 таблиц:

| Таблица | Описание |
| --- | --- |
| Table 1 | Номер плавки, дата, вес стали, параметры полупродукта из дуговой печи (Last EOP) |
| Table 2 | Целевые значения элементов (Min / Target / Max) и фактический состав стали |
| Table 3 | Лог событий с временными метками |
| Table 4, 6, 9 | Количества добавляемых ферросплавов на установке печь-ковш |
| Table 5 | Температура перед сливом |
| Table 7 | Данные о смене и руководителе |
| Table 8 | Металлолом, добавляемый на разных стадиях |
| Table 10 | Металлолом в печь-ковш |

**Важно:**  
> Столбцы C–I из Table 1 используются заказчиком для идентификации материала и не используются в нашей работе.

### 🛠️ Задачи проекта

1. Очистка и предобработка 10 таблиц (работа с пропусками, дубликатами, выбросами)
2. Feature Engineering – создание признаков для модели
3. Обучение модели для предсказания необходимого количества ферросплавов
4. Формирование рекомендаций для сталевара

### ▶️ Результат

Модель, которая советует легирующие добавки, чтобы фактическое содержание оказалось в целевом диапазоне.
